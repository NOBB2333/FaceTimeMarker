新增截图时：

1. 放到 `docs/assert/`。
2. 文件名用中文业务名或清晰英文名。
3. 在对应文档中用相对路径引用。
4. 如果替换旧截图，确认 README 和用户手册里的说明仍然准确。

## 维护规则

- 新功能上线：同步用户手册和产品设计。
- 新 API/表/配置：同步技术架构和开发指南。
- 前端页面变化：同步截图和页面说明。
- 识别调参变化：同步用户手册、技术文档和配置注释。
- 路线图完成项：同步项目完整性评估和实施路线图。
